Part 4 - Detail Page

  • Lets use get to find a restaurant by name
  • Now lets use django models.
  • Restaurant.objects.get(FIELD=VALUE)
  • ex. Restaurant.objects.get(name="mcdonalds")
  • returns a SINGLE restaurant where that object has that value
  • Use this for the detail page
  • https://docs.djangoproject.com/en/1.8/topics/db/queries/#retrieving-a-single-object-with-get

Part 5 - Rating

  • Lets use filter to find restaurants by ratings
  • Restaurant.objects.filter(FIELD=VALUE)
  • returns multiple restaurants where that object has that value
  • We can also use FIELD__gte and FIELD__lte for greater than or less than or equal to
  • Use this for the ratings page
  • https://docs.djangoproject.com/en/1.8/topics/db/queries/#retrieving-specific-objects-with-filters
