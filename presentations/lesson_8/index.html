Migrations

When you change a model, also make and run migrations


python manage.py makemigrations
python manage.py migrate
  • Migrations create changes in your database
  • python manage.py sqlmigrate restaurant 0001

  • {} - context dictionary to send to template

urls

  • Make urls.py in your app (restaurants) folder
    • 
from django.conf.urls import url

from . import views

urlpatterns = [
    url(r'^$', views.index, name='index'),
]
  • In your project folder's urls.py
    • 
from django.conf.urls import include, url
from django.contrib import admin

urlpatterns = [
    url(r'^restaurants/', include('restaurants.urls')),
    url(r'^admin/', include(admin.site.urls)),
]
